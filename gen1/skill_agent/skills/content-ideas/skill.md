# Content Ideas Generator

## Protocol

1.  **Understand Goal**: Ask the user for the topic, target audience, and desired platform (e.g., blog, social media, video).
2.  **Brainstorm Core Concepts**: Based on the input, generate 3-5 high-level content concepts.
3.  **Elaborate on Concepts**: For each core concept, provide 2-3 specific content ideas, including a catchy title, a brief description, and potential keywords/hashtags.
4.  **Offer Variations**: Suggest different angles or formats for the ideas (e.g., listicle, how-to guide, video tutorial, infographic).
5.  **Refine and Present**: Present the ideas clearly, allowing the user to pick and choose or request further refinement.

## Output Format

```
## Content Ideas for [Topic] on [Platform]

### Core Concept 1: [Concept Title]
- **Idea 1.1**: "[Catchy Title]"
  - *Description*: [Brief description of the content idea.]
  - *Keywords/Hashtags*: [comma-separated list]
- **Idea 1.2**: "[Catchy Title]"
  - *Description*: [Brief description of the content idea.]
  - *Keywords/Hashtags*: [comma-separated list]

### Core Concept 2: [Concept Title]
- **Idea 2.1**: "[Catchy Title]"
  - *Description*: [Brief description of the content idea.]
  - *Keywords/Hashtags*: [comma-separated list]
- **Idea 2.2**: "[Catchy Title]"
  - *Description*: [Brief description of the content idea.]
  - *Keywords/Hashtags*: [comma-separated list]
```
